构建产物清单

每次 main 分支构建成功后会创建一个 GitHub Release，tag 是 build-<SHA>。Connector 可以下载单二进制运行，也可以下载 Docker 镜像包运行。

GitHub Release: build-<SHA>
  testflying-connector-windows-amd64-<SHA>.zip       # Windows 单二进制
  testflying-connector-linux-amd64-<SHA>.tar.gz      # Linux 单二进制
  testflying-connector-darwin-arm64-<SHA>.tar.gz     # macOS Apple Silicon 单二进制
  testflying-connector-darwin-amd64-<SHA>.tar.gz     # macOS Intel 单二进制
  testflying-connector-<SHA>.tar.gz                  # Connector Docker 镜像
  testflying-server-<SHA>.tar.gz                     # 中心后台 API 镜像
  testflying_server-0.1.0-py3-none-any.whl           # 后端 wheel

Actions artifact 备用入口:
  testflying-build-<SHA>/
    connectors/testflying-connector-windows-amd64-<SHA>.zip
    images/testflying-connector-<SHA>.tar.gz

Windows：下载 exe 并创建长期运行任务

$Sha = "<commit sha>"
$Root = "C:\testflying-connector"
$TaskName = "testflying-connector-{{ account.id }}"

{% if account_store_platform == "ios" %}
New-Item -ItemType Directory -Force $Root, "$Root\secrets\appstoreconnect" | Out-Null
{% elif account_store_platform == "android" %}
New-Item -ItemType Directory -Force $Root, "$Root\secrets\googleplay" | Out-Null
{% else %}
New-Item -ItemType Directory -Force $Root, "$Root\secrets\appstoreconnect", "$Root\secrets\googleplay" | Out-Null
{% endif %}

gh release download "build-$Sha" `
  -R baoluchuling/testflying-api `
  -p "testflying-connector-windows-amd64-$Sha.zip" `
  -D $Root

Expand-Archive -Force "$Root\testflying-connector-windows-amd64-$Sha.zip" $Root
Move-Item -Force "$Root\testflying-connector-windows-amd64-$Sha.exe" "$Root\testflying-connector.exe"

@"
`$env:TESTFLYING_CONNECTOR_LISTEN_ADDR = ":8100"
`$env:TESTFLYING_CONNECTOR_STORE_MODE = "live"
`$env:TESTFLYING_CONNECTOR_DEVELOPER_ACCOUNT_ID = "{{ account.id }}"
`$env:TESTFLYING_CONNECTOR_TOKEN = "<random-token>"
{% if account_store_platform in ["ios", "mixed"] %}
# App Store Connect API Key，把 .p8 放到 $Root\secrets\appstoreconnect
`$env:TESTFLYING_CONNECTOR_APPLE_ISSUER_ID = "<issuer-id>"
`$env:TESTFLYING_CONNECTOR_APPLE_KEY_ID = "<key-id>"
`$env:TESTFLYING_CONNECTOR_APPLE_PRIVATE_KEY_PATH = "$Root\secrets\appstoreconnect\AuthKey_XXXX.p8"
{% endif %}
{% if account_store_platform in ["android", "mixed"] %}
# Google Play service account，把 JSON 放到 $Root\secrets\googleplay
`$env:TESTFLYING_CONNECTOR_GOOGLE_SERVICE_ACCOUNT_JSON_PATH = "$Root\secrets\googleplay\service-account.json"
{% endif %}

& "$Root\testflying-connector.exe"
"@ | Set-Content -Encoding UTF8 "$Root\run-connector.ps1"

schtasks /Create /TN $TaskName /SC ONSTART /RL HIGHEST /RU SYSTEM /TR "powershell.exe -ExecutionPolicy Bypass -File $Root\run-connector.ps1" /F
schtasks /Run /TN $TaskName
Invoke-RestMethod http://127.0.0.1:8100/health

Docker：从 GitHub Release 下载并加载 Connector 镜像

SHA=<commit sha>

mkdir -p /opt/testflying/artifacts
cd /opt/testflying/artifacts

gh release download "build-$SHA" \
  -R baoluchuling/testflying-api \
  -p "testflying-connector-$SHA.tar.gz" \
  -D .

gunzip -c "testflying-connector-$SHA.tar.gz" | docker load
docker tag "testflying-connector:$SHA" testflying-connector:local

备用：从 GitHub Actions artifact 下载

RUN_ID=<GitHub Actions run id>
SHA=<commit sha>

gh run download "$RUN_ID" \
  -R baoluchuling/testflying-api \
  -D .

ls -R "testflying-build-$SHA"
gunzip -c "testflying-build-$SHA/images/testflying-connector-$SHA.tar.gz" | docker load
docker tag "testflying-connector:$SHA" testflying-connector:local
